Ekran Koordinat Sistemi

Raylib'de 2D çizimde koordinat sistemi şöyle çalışır:

  • (0, 0) ekranın sol üst köşesidir
  • X sağa doğru artar
  • Y aşağı doğru artar (matematik derslerindeki Y yukarı doğru artar — burada TAM TERSİ!)
  • Birim: piksel
⚠️ Dikkat: Y Ekseni Ters!

Ekran koordinatlarında Y aşağı artar. Yani y = 0 en üst, y = 599 en alttır (600 piksellik pencerede). Bu başlangıçta kafa karıştırıcı olabilir ama tüm grafik API'lerinde böyledir.

Dikdörtgenler

// En basit: x, y, genislik, yukseklik, renk
DrawRectangle(50, 100, 200, 80, RED);

// Vector2 ile
DrawRectangleV((Vector2){50, 100}, (Vector2){200, 80}, RED);

// Rectangle struct ile
Rectangle kutu = { 50, 100, 200, 80 };
DrawRectangleRec(kutu, RED);

// Sadece kenarlar
DrawRectangleLines(50, 100, 200, 80, BLUE);
DrawRectangleLinesEx(kutu, 3.0f, BLUE);  // 3 piksel kalinlik

// Yuvarlatilmis koseler
DrawRectangleRounded(kutu, 0.3f, 4, GREEN);
// 0.3f = yuvarlatma orani (0=kose, 1=tamamen yuvarlak)
// 4 = segment sayisi (kalite)

// Renk gecisi (gradient)
DrawRectangleGradientH(50, 200, 200, 50, RED, BLUE);  // yatay
DrawRectangleGradientV(50, 260, 200, 50, RED, BLUE);  // dikey
🧪 Deneyin: Rounded Değeri

DrawRectangleRounded'da 0.3f değerini 0.0f (keskin köşe) ve 1.0f (tamamen yuvarlak) yaparak farkı görün.

Daireler

// Merkez (x,y), yaricap, renk
DrawCircle(200, 300, 50, BLUE);

// Vector2 ile
DrawCircleV((Vector2){200, 300}, 50, BLUE);

// Sadece kenar
DrawCircleLines(200, 300, 50, DARKBLUE);

// Pasta dilimi (sector)
DrawCircleSector((Vector2){400, 300}, 60, 0, 120, 16, ORANGE);
// baslangic acisi=0, bitis=120 derece, 16 segment

// Halka (ring)
DrawRing((Vector2){600, 300}, 30, 50, 0, 270, 16, PURPLE);
// ic yaricap=30, dis yaricap=50

Çizgiler

// Basit cizgi
DrawLine(10, 10, 200, 150, RED);

// Vector2 ile
DrawLineV((Vector2){10,10}, (Vector2){200,150}, RED);

// Kalinlik ayarli
DrawLineEx((Vector2){10,10}, (Vector2){200,150}, 3.0f, BLUE);

// Bezier egrisi (yumusak egri)
DrawLineBezier((Vector2){50, 400}, (Vector2){350, 350}, 2.0f, DARKGREEN);

Üçgen ve Poligon

// Ucgen: 3 kose noktasi (saat yonunde!)
DrawTriangle(
    (Vector2){100, 300},  // ust
    (Vector2){50, 400},   // sol alt
    (Vector2){150, 400},  // sag alt
    GOLD
);

// Duzgun poligon: merkez, kenar sayisi, yaricap, donme, renk
DrawPoly((Vector2){300, 350}, 6, 40, 0, VIOLET);   // altigen
DrawPoly((Vector2){450, 350}, 5, 40, 0, SKYBLUE);  // besgen
DrawPoly((Vector2){600, 350}, 8, 40, 0, ORANGE);   // sekizgen
🧪 Deneyin: Kenar Sayısı

DrawPoly'daki kenar sayısını 3 (üçgen), 4 (kare), 12 (neredeyse daire), 32 (daire gibi) yaparak şeklin nasıl değiştiğini gözlemleyin.

Yazı (Text)

// Basit yazi
DrawText("Merhaba Dunya!", 100, 50, 20, DARKGRAY);
// metin, x, y, font boyutu, renk

// Degisken gosterme
int skor = 42;
DrawText(TextFormat("Skor: %d", skor), 100, 80, 20, RED);
// %d = int, %.2f = 2 ondalik float, %s = string

// Yazi genisligi olcme (merkezleme icin)
const char *metin = "Ortala Beni";
int genislik = MeasureText(metin, 24);
DrawText(metin, (800 - genislik) / 2, 200, 24, BLACK);
// (800 - genislik) / 2 = yatay merkez

Doku (Texture)

// Dosyadan yukleme
Texture2D resim = LoadTexture("karakter.png");  // baslatmada 1 kez
DrawTexture(resim, 100, 100, WHITE);            // her karede ciz
UnloadTexture(resim);                           // cikiste temizle

// Procedural (kod ile) olusturma
Image img = GenImageChecked(64, 64, 8, 8, ORANGE, WHITE);
Texture2D doku = LoadTextureFromImage(img);
UnloadImage(img);  // Image CPU'da, artik gerekmez
// doku GPU'da yasayor - cizerken kullaniyoruz
ℹ️ Bilgi: Image vs Texture2D

Image = CPU belleğinde (RAM). Piksel piksel düzenleyebilirsiniz.
Texture2D = GPU belleğinde (VRAM). Çizmek için gerekli.
Akış: dosya → Image → Texture2D → çiz. Image işiniz bittikten sonra UnloadImage ile silebilirsiniz.

Çarpışma Testi (Collision)

Rectangle kutu1 = { 100, 100, 50, 50 };
Rectangle kutu2 = { 130, 120, 60, 40 };

// Iki dikdortgen carpisiyor mu?
if (CheckCollisionRecs(kutu1, kutu2)) {
    DrawText("CARPISMA!", 100, 50, 20, RED);
}

// Daire + dikdortgen
Vector2 daireMerkez = { 200, 200 };
float daireR = 25;
if (CheckCollisionCircleRec(daireMerkez, daireR, kutu1)) { ... }

// Nokta + dikdortgen (fare tiklama icin ideal)
Vector2 fare = GetMousePosition();
if (CheckCollisionPointRec(fare, kutu1)) {
    // fare kutunun uzerinde!
}
🧪 Deneyin: Fare ile Çarpışma

Fare konumunu bir daire olarak kullanın ve CheckCollisionCircleRec ile bir kutuya çarptığında rengini değiştirin. Bu, buton yapmanın temel mantığıdır!

Örnek Kod

🎮 Raylib: ornek09_2d_cizim.c

Tüm 2D çizim fonksiyonlarını tek ekranda gösterir. Fare ile çarpışma testi yapar.

#include "raylib.h"
#include <math.h>

int main(void)
{
    const int W = 950, H = 650;
    InitWindow(W, H, "Ornek 09 - 2D Cizim");
    SetTargetFPS(60);
    int frame = 0;

    Vector2 dairePos = { 700.0f, 500.0f };
    float daireR = 30.0f;
    Rectangle sabitKutu = { 600, 460, 120, 80 };
    bool carpisiyor = false;

    Image img = GenImageChecked(64, 64, 8, 8, ORANGE, WHITE);
    Texture2D doku = LoadTextureFromImage(img);
    UnloadImage(img);

    while (!WindowShouldClose())
    {
        frame++;
        float t = frame * 0.03f;
        dairePos = GetMousePosition();
        carpisiyor = CheckCollisionCircleRec(dairePos, daireR, sabitKutu);

        BeginDrawing();
        ClearBackground(RAYWHITE);

        // Dikdortgenler
        DrawRectangle(20, 80, 80, 50, RED);
        DrawRectangleRounded((Rectangle){220, 80, 80, 50}, 0.3f, 4, GREEN);
        DrawRectangleGradientH(320, 80, 80, 50, SKYBLUE, DARKPURPLE);

        // Daireler
        DrawCircle(60, 220, 30, MAROON);
        DrawCircleSector((Vector2){260, 220}, 30, 0, 120 + sinf(t)*60, 16, ORANGE);

        // Poligon (donuyor)
        DrawPoly((Vector2){180, 450}, 6, 35, t * 30, VIOLET);

        // Doku
        DrawTexture(doku, 450, 215, WHITE);
        DrawTextureEx(doku, (Vector2){550, 215}, sinf(t)*15, 1.5f, SKYBLUE);

        // Carpisma testi
        Color kRenk = carpisiyor ? RED : Fade(SKYBLUE, 0.5f);
        DrawRectangleRec(sabitKutu, kRenk);
        DrawCircleV(dairePos, daireR, Fade(carpisiyor ? RED : GREEN, 0.6f));
        if (carpisiyor) DrawText("CARPISMA!", 620, 555, 20, RED);

        EndDrawing();
    }
    UnloadTexture(doku);
    CloseWindow();
    return 0;
}

Önemli Satırlar

GenImageChecked(64, 64, 8, 8, ORANGE, WHITE)

64x64 piksellik bir dama tahtası deseni oluşturur. Her kare 8x8 piksel. Dosya yüklemeden doku oluşturmanın en kolay yolu.

CheckCollisionCircleRec(dairePos, daireR, sabitKutu)

Dairenin kutunun içine girip girmediğini kontrol eder. true dönerse çarpışma var — rengi kırmızıya çeviriyoruz.

sinf(t) * 60

Sinüs fonksiyonu -1 ile 1 arası salınım yapar. * 60 ile -60..+60 arası değer elde ederiz. Bunu açıya ekleyerek pasta dilimini canlandırıyoruz.

🧪 Deneyin: Doku Döndürme Hızı

sinf(t)*15 ifadesindeki 15 değerini 45 veya 90 yapın — doku çok daha çılgınca döner!
