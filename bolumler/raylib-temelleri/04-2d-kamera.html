Neden Kamera Lazım?

Pencereniz 900×600 piksel. Ama dünyanız 10.000×10.000 birim olabilir — bir şehir, bir harita, devasa bir seviye. Tamamını aynı anda gösteremezsiniz. Kamera, bu büyük dünyanın sadece bir kısmını pencerede gösteren "penceredir".

ℹ️ Bilgi: Dünya vs Ekran Koordinatları

Dünya koordinatları: Nesnelerin gerçek konumu (x=5000 olabilir).
Ekran koordinatları: Pikseller (0..899, 0..599).
Kamera bu dönüşümü yapar: dünya koordinatını alır, ekranda nereye düşeceğini hesaplar.

Camera2D Yapısı

Camera2D kamera = { 0 };  // tum alanlari sifirla

kamera.target = (Vector2){ 500, 300 };   // dunya noktasi
kamera.offset = (Vector2){ 450, 300 };   // ekran pikseli
kamera.rotation = 0.0f;                  // derece
kamera.zoom = 1.0f;                      // carpan

target — Kameranın Baktığı Dünya Noktası

Dünya koordinatlarında bir nokta. Kamera bu noktayı "takip eder". Genelde oyuncunun merkezi buraya atanır.

offset — Bu Noktanın Ekranda Düşeceği Yer

Piksel cinsinden. Genelde ekranın ortası: {ekranGenislik/2, ekranYukseklik/2}. Bu sayede target noktası ekranın tam ortasında görünür.

zoom — Yakınlaştırma

1.0 = normal, 2.0 = 2x yakın (nesneler büyük), 0.5 = uzak (nesneler küçük, daha geniş alan görünür).

rotation — Döndürme

Derece cinsinden. 45 = dünya 45° döndürülmüş görünür.

BeginMode2D / EndMode2D

Bu blok içindeki çizimler dünya koordinatlarında yapılır — kamera onları ekrana taşır. Blok dışındaki çizimler ekran koordinatlarında yapılır (UI için).

BeginDrawing();
    ClearBackground(RAYWHITE);

    BeginMode2D(kamera);
        // --- DUNYA KOORDINATLARI ---
        // Buradaki x=5000 bile olsa kamera dogru yere cizer
        DrawRectangle(5000, 200, 100, 100, RED);
        DrawCircle(-3000, 100, 50, BLUE);
    EndMode2D();

    // --- EKRAN KOORDINATLARI (UI) ---
    // Skor tablosu, saglik barı vs. burada
    DrawText("Skor: 42", 10, 10, 20, BLACK);
EndDrawing();
⚠️ Dikkat: UI'yı Mode2D İçinde Çizmeyin!

Mode2D içindeki çizimler kamerayla birlikte hareket eder. Skor tablosu veya buton gibi sabit kalması gereken şeyleri EndMode2D'den sonra çizin.

Dünya-Ekran Dönüşümleri

// Fare ekran pikselinden dunya koordinatina
Vector2 fareDunya = GetScreenToWorld2D(GetMousePosition(), kamera);

// Dunya noktasindan ekran pikseline
Vector2 ekranPos = GetWorldToScreen2D((Vector2){5000, 200}, kamera);

Bu fonksiyonlar kritiktir: fare tıklamasının dünyada nereye denk geldiğini bulmak için GetScreenToWorld2D kullanırsınız.

Örnek Kod: 2D Şehir ve Kamera

🎮 Raylib: ornek10_2d_kamera.c

WASD ile oyuncuyu taşıyın, kamera takip eder. Q/E ile döndürün, tekerlek ile zoom yapın.

#include "raylib.h"

#define BINA_SAYISI 50

int main(void)
{
    const int W = 900, H = 600;
    InitWindow(W, H, "Ornek 10 - 2D Kamera");
    SetTargetFPS(60);

    Rectangle oyuncu = { 400, 280, 30, 50 };
    float oyuncuHiz = 5.0f;

    // Rastgele binalar olustur
    Rectangle binalar[BINA_SAYISI];
    Color binaRenkleri[BINA_SAYISI];
    int ofs = 0;
    for (int i = 0; i < BINA_SAYISI; i++) {
        int en = GetRandomValue(60, 160);
        int boy = GetRandomValue(80, 350);
        binalar[i] = (Rectangle){ ofs - 4000, H - 100 - boy, en, boy };
        binaRenkleri[i] = (Color){
            GetRandomValue(120,250),
            GetRandomValue(120,250),
            GetRandomValue(120,250), 255 };
        ofs += en + GetRandomValue(2, 12);
    }

    // Kamera ayarla
    Camera2D kamera = { 0 };
    kamera.target = (Vector2){ oyuncu.x + 15, oyuncu.y + 25 };
    kamera.offset = (Vector2){ W/2.0f, H/2.0f };
    kamera.zoom = 1.0f;

    while (!WindowShouldClose())
    {
        // Oyuncu hareketi (DUNYA koordinatlarinda)
        if (IsKeyDown(KEY_D)) oyuncu.x += oyuncuHiz;
        if (IsKeyDown(KEY_A)) oyuncu.x -= oyuncuHiz;
        if (IsKeyDown(KEY_W)) oyuncu.y -= oyuncuHiz;
        if (IsKeyDown(KEY_S)) oyuncu.y += oyuncuHiz;

        // Kamera: oyuncuyu takip et
        kamera.target = (Vector2){ oyuncu.x + 15, oyuncu.y + 25 };

        // Kamera donme ve zoom
        if (IsKeyDown(KEY_Q)) kamera.rotation -= 1.5f;
        if (IsKeyDown(KEY_E)) kamera.rotation += 1.5f;
        if (IsKeyPressed(KEY_R)) kamera.rotation = 0;
        kamera.zoom += GetMouseWheelMove() * 0.15f;
        if (kamera.zoom < 0.2f) kamera.zoom = 0.2f;
        if (kamera.zoom > 5.0f) kamera.zoom = 5.0f;

        BeginDrawing();
            ClearBackground((Color){200, 220, 240, 255});

            BeginMode2D(kamera);
                // Zemin
                DrawRectangle(-5000, H-100, 10000, 200, DARKGREEN);
                // Binalar
                for (int i = 0; i < BINA_SAYISI; i++)
                    DrawRectangleRec(binalar[i], binaRenkleri[i]);
                // Oyuncu
                DrawRectangleRec(oyuncu, RED);
            EndMode2D();

            // UI (ekran koordinatlarinda - kameradan bagimsiz)
            DrawText(TextFormat("Zoom: %.2f", kamera.zoom), 10, 10, 16, BLACK);

            // Fare dunya koordinati
            Vector2 fd = GetScreenToWorld2D(GetMousePosition(), kamera);
            DrawText(TextFormat("Dunya: (%.0f, %.0f)", fd.x, fd.y), 10, 30, 14, GRAY);
        EndDrawing();
    }
    CloseWindow();
    return 0;
}

Önemli Noktalar

kamera.target = oyuncu merkezi

Her karede kameranın hedefini oyuncuya eşitliyoruz. Bu sayede oyuncu hareket ettikçe kamera onu takip eder.

kamera.offset = { W/2, H/2 }

Target noktasının ekranın TAM ORTASINDA görünmesini sağlar. Eğer {0, 0} yaparsanız, oyuncu sol üst köşede kalır.

GetScreenToWorld2D

Fare piksel koordinatını dünya koordinatına çevirir. Kameranın zoom ve rotation'ını hesaba katar.

🧪 Deneyin: Offset'i Değiştirin

kamera.offset'i (Vector2){0, 0} yapın. Oyuncu artık ekranın sol üst köşesinde — çünkü target noktası (0,0) pikselinde gösterilir.

🧪 Deneyin: Zoom Başlangıcı

kamera.zoom = 0.3f; ile başlayın — tüm şehir görünür. 3.0f ile başlayın — sadece yakın bölge.

🧪 Deneyin: Yumuşak Takip

Direkt atama yerine: kamera.target.x += (oyuncu.x - kamera.target.x) * 0.05f; — kamera yavaşça yakalar (smooth follow).
