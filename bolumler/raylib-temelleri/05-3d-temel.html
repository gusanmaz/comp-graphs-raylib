2D'den 3D'ye Geçiş

2D'de her şey x (yatay) ve y (dikey) ile tanımlanıyordu. 3D'de bir eksen daha ekleniyor: z (derinlik). Bu üç eksenle uzaydaki herhangi bir noktayı tanımlayabilirsiniz.

Raylib'in 3D eksen sistemi:

  • X = sağa doğru (kırmızı)
  • Y = yukarı doğru (yeşil)
  • Z = kameraya doğru / ekrandan dışarı (mavi)
ℹ️ Bilgi: Neden Eksenler Önemli?

Her 3D motor farklı eksen sistemi kullanabilir. Unity'de Y yukarı, Unreal'de Z yukarıdır. Raylib'de Y yukarıdır. Bunu bilmek, nesneleri doğru konumlandırmak için şarttır.

Camera3D Yapısı

Camera3D kamera = { 0 };
kamera.position   = (Vector3){ 6.0f, 6.0f, 6.0f };  // kameranin KONUMU
kamera.target     = (Vector3){ 0.0f, 0.5f, 0.0f };  // BAKTIĞI nokta
kamera.up         = (Vector3){ 0.0f, 1.0f, 0.0f };  // yukari yon
kamera.fovy       = 50.0f;                           // gorus acisi
kamera.projection = CAMERA_PERSPECTIVE;              // projeksiyon turu

position — Kameranın 3D Konumu

Kamera uzayda nerede duruyor? {6, 6, 6} = x=6, y=6, z=6 noktasında — sahneye "çapraz yukarıdan" bakıyor.

target — Kameranın Baktığı Nokta

Kamera hangi noktaya odaklanıyor? {0, 0.5, 0} = orijinin biraz üstüne bakıyor.

up — Yukarı Yön

Kameranın "tepesi" hangi yöne bakıyor? Genelde {0, 1, 0} (Y yukarı). Bunu değiştirirseniz kamera "yana yatmış" gibi görünür.

fovy — Görüş Açısı (Field of View)

Derece cinsinden. Daha büyük = daha geniş görüş (ama distorsiyon artar). Tipik değerler: 45°-60°.

Projeksiyon Türleri

3D dünyayı 2D ekrana aktarmanın iki yolu var:

Perspektif (CAMERA_PERSPECTIVE)

  • Uzak nesneler küçük, yakın nesneler büyük görünür
  • Gerçek hayattaki gibi — derinlik hissi verir
  • fovy = görüş açısı (derece)
  • Oyunlar, simülasyonlar, gerçekçi render için

Ortografik (CAMERA_ORTHOGRAPHIC)

  • Uzak ve yakın nesneler aynı boyutta görünür
  • Derinlik algısı yok — teknik çizim gibi
  • fovy = görünür alanın boyutu (birim cinsinden)
  • 2.5D oyunlar, harita görünümü, CAD programları için

Örnek Kod: Perspektif vs Ortografik

🎮 Raylib: ornek11_3d_temel.c

Ekranın sol yarısı perspektif, sağ yarısı ortografik projeksiyon gösterir. SPACE ile otomatik dönme, UP/DOWN ile FOV ayarı.

#include "raylib.h"
#include <math.h>

int main(void)
{
    const int W = 1000, H = 600;
    InitWindow(W, H, "Ornek 11 - 3D Temel ve Projeksiyon");
    SetTargetFPS(60);

    // Perspektif kamera
    Camera3D kamP = { {6,6,6}, {0,0.5f,0}, {0,1,0}, 50, CAMERA_PERSPECTIVE };
    // Ortografik kamera
    Camera3D kamO = { {6,6,6}, {0,0.5f,0}, {0,1,0}, 12, CAMERA_ORTHOGRAPHIC };

    float donme = 0.0f;
    bool otoDon = true;

    while (!WindowShouldClose())
    {
        if (IsKeyPressed(KEY_SPACE)) otoDon = !otoDon;
        if (otoDon) donme += 0.5f;

        // FOV ayari
        if (IsKeyDown(KEY_UP))   kamP.fovy -= 0.5f;
        if (IsKeyDown(KEY_DOWN)) kamP.fovy += 0.5f;
        if (kamP.fovy < 10) kamP.fovy = 10;
        if (kamP.fovy > 120) kamP.fovy = 120;

        // Kamerayi daire uzerinde dondur
        float rad = donme * DEG2RAD;
        kamP.position = (Vector3){ cosf(rad)*8, 5, sinf(rad)*8 };
        kamO.position = kamP.position;

        BeginDrawing();
            ClearBackground(RAYWHITE);

            // Sol yari: PERSPEKTIF
            BeginScissorMode(0, 0, W/2, H);
            BeginMode3D(kamP);
                DrawGrid(10, 1.0f);
                DrawLine3D((Vector3){0,0,0}, (Vector3){3,0,0}, RED);
                DrawLine3D((Vector3){0,0,0}, (Vector3){0,3,0}, GREEN);
                DrawLine3D((Vector3){0,0,0}, (Vector3){0,0,3}, BLUE);
                DrawCube((Vector3){0,0.5f,0}, 1,1,1, Fade(RED, 0.7f));
                DrawCubeWires((Vector3){0,0.5f,0}, 1,1,1, MAROON);
                DrawSphere((Vector3){2,0.5f,0}, 0.5f, Fade(BLUE, 0.7f));
                DrawCylinder((Vector3){-2,0,0}, 0.5f,0.5f,1.5f,12, Fade(GREEN, 0.7f));
            EndMode3D();
            EndScissorMode();

            // Sag yari: ORTOGRAFIK
            BeginScissorMode(W/2, 0, W/2, H);
            ClearBackground((Color){245,245,250,255});
            BeginMode3D(kamO);
                DrawGrid(10, 1.0f);
                DrawCube((Vector3){0,0.5f,0}, 1,1,1, Fade(RED, 0.7f));
                DrawSphere((Vector3){2,0.5f,0}, 0.5f, Fade(BLUE, 0.7f));
                DrawCylinder((Vector3){-2,0,0}, 0.5f,0.5f,1.5f,12, Fade(GREEN, 0.7f));
            EndMode3D();
            EndScissorMode();

            // Etiketler
            DrawLine(W/2, 0, W/2, H, DARKGRAY);
            DrawRectangle(10,10,200,28, Fade(BLACK,0.6f));
            DrawText("PERSPEKTIF", 20, 15, 18, WHITE);
            DrawRectangle(W/2+10,10,200,28, Fade(BLACK,0.6f));
            DrawText("ORTOGRAFIK", W/2+20, 15, 18, WHITE);
            DrawText(TextFormat("FOV: %.0f  SPACE:don", kamP.fovy), 10, H-25, 13, GRAY);
        EndDrawing();
    }
    CloseWindow();
    return 0;
}

Önemli Kavramlar

BeginScissorMode(x, y, w, h)

Ekranın belirli bir dikdörtgen alanına çizimi sınırlar. Burada ekranı ikiye bölmek için kullanıyoruz — sol yarı perspektif, sağ yarı ortografik.

DEG2RAD

Derece → radyan dönüşüm sabiti. cosf() ve sinf() radyan bekler. donme * DEG2RAD ile dereceyi radyana çeviririz.

cosf(rad)*8, 5, sinf(rad)*8

Kamerayı orijin etrafında daire üzerinde döndürür. cos/sin bir daire çizer (yarıçap=8), y=5 sabit kalır.

🧪 Deneyin: FOV Değiştirin

UP/DOWN ile perspektif FOV'u değiştirin. 20° = dar/tele lens (nesneler yakın ve düz), 100° = geniş açı (balık gözü etkisi, kenarlar eğrilir).

🧪 Deneyin: Kamera Yüksekliği

kamP.position satırında y=5 değerini y=1 yapın — neredeyse zemin seviyesinden bakarsınız. y=15 yapın — kuş bakışı.

🧪 Deneyin: Ortografik Boyut

kamO.fovy = 12 değerini 5 yapın — yakınlaşır. 30 yapın — tüm sahne küçücük görünür. Perspektifteki zoom'un ortografik karşılığı budur.
