UpdateCamera Fonksiyonu

Önceki bölümde kamerayı elle konumlandırdık. Ama raylib size hazır kamera kontrolleri sunar — tek satırla!

UpdateCamera(&kamera, CAMERA_ORBITAL);

Bu fonksiyon her karede çağrılır ve kameranın position, target değerlerini fare + klavye girdisine göre otomatik günceller.

ℹ️ Bilgi: Neden &kamera?

C'de bir fonksiyona struct gönderdiğinizde, kopya gönderilir — orijinal değişmez. & (adres operatörü) ile struct'ın kendisini (adresini) gönderirsiniz, böylece fonksiyon orijinali değiştirebilir. Bu "pointer" kavramıdır.

4 Kamera Modu

CAMERA_FREE — Serbest Uçuş

  • WASD: ileri/geri/sağ/sol hareket
  • Fare: bakış yönü
  • Scroll: yakınlaş/uzaklaş
  • Kullanım: Sahne editörleri, debug modu, serbest keşif

Kamera uzayda serbestçe "uçar". Hiçbir kısıtlama yok — yerçekimi de yok.

CAMERA_ORBITAL — Yörünge

  • Kamera, target noktasının etrafında otomatik döner
  • Scroll: yakınlaş/uzaklaş
  • Kullanım: Ürün gösterimi, 3D model inceleme, vitrin

Hedefin etrafında sürekli döner. Bir nesneyi her açıdan incelemek için ideal.

CAMERA_FIRST_PERSON — Birinci Şahıs (FPS)

  • WASD: yürüme (zeminde)
  • Fare: bakış yönü
  • Kullanım: FPS oyunlar, mimari yürüyüş

position = "gözünüzün" konumu. Fare ile etrafınıza bakarsınız, WASD ile yürürsünüz.

CAMERA_THIRD_PERSON — Üçüncü Şahıs

  • Kamera karakterin arkasından bakar
  • target = karakterin konumu
  • Kullanım: RPG, aksiyon oyunları

Karakter önde yürür, kamera arkasından takip eder.

Fare İmleci Kontrolü

DisableCursor();  // Fareyi gizle ve pencereye kilitle
EnableCursor();   // Fareyi geri getir

// Programda M ile toggle:
if (IsKeyPressed(KEY_M)) {
    if (IsCursorHidden()) EnableCursor();
    else DisableCursor();
}
ℹ️ Bilgi: Neden Fare Gizlenir?

FREE ve FIRST_PERSON modlarında fare hareketi bakış yönünü kontrol eder. Fare görünürse ve pencere kenarına gelirse, dönme durur. DisableCursor() fareyi "sonsuz" yapar — ne kadar hareket ederseniz edin kenar sorunu olmaz.

Örnek Kod: 4 Mod Arası Geçiş

🎮 Raylib: ornek12_3d_kamera.c

1-4 tuşları ile kamera modunu değiştirin. M ile fareyi açın/kapatın.

#include "raylib.h"

int main(void)
{
    const int W = 950, H = 600;
    InitWindow(W, H, "Ornek 12 - 3D Kamera Modlari");

    Camera3D kam = { {5,5,5}, {0,0,0}, {0,1,0}, 55, CAMERA_PERSPECTIVE };

    int mi = 0;  // aktif mod indexi
    int modlar[] = { CAMERA_FREE, CAMERA_ORBITAL,
                     CAMERA_FIRST_PERSON, CAMERA_THIRD_PERSON };
    const char *ad[] = { "FREE", "ORBITAL", "FIRST_PERSON", "THIRD_PERSON" };

    // Sahne nesneleri
    Vector3 kp[] = { {0,0.5f,0},{3,0.5f,0},{-3,0.5f,0},
                     {0,0.5f,3},{0,0.5f,-3},{3,1,3},{-3,0.5f,-3} };
    Color kr[] = { RED, BLUE, GREEN, ORANGE, PURPLE, GOLD, SKYBLUE };

    DisableCursor();  // Baslangicta fare gizli
    SetTargetFPS(60);

    while (!WindowShouldClose())
    {
        // Mod degistir
        if (IsKeyPressed(KEY_ONE))   { mi=0; kam.position=(Vector3){5,5,5}; }
        if (IsKeyPressed(KEY_TWO))   { mi=1; kam.position=(Vector3){5,5,5}; }
        if (IsKeyPressed(KEY_THREE)) {
            mi=2;
            kam.position=(Vector3){0,2,0};
            kam.target=(Vector3){0,2,1};
        }
        if (IsKeyPressed(KEY_FOUR))  {
            mi=3;
            kam.position=(Vector3){0,2,-5};
            kam.target=(Vector3){0,0,0};
        }

        // Fare toggle
        if (IsKeyPressed(KEY_M)) {
            if (IsCursorHidden()) EnableCursor();
            else DisableCursor();
        }

        // KRITIK: Kamerayi guncelle
        UpdateCamera(&kam, modlar[mi]);

        BeginDrawing();
            ClearBackground((Color){180,200,220,255});

            BeginMode3D(kam);
                DrawPlane((Vector3){0,0,0}, (Vector2){30,30}, Fade(DARKGREEN,0.6f));
                DrawGrid(30, 1.0f);

                // Eksenler
                DrawLine3D((Vector3){0,0,0}, (Vector3){5,0,0}, RED);
                DrawLine3D((Vector3){0,0,0}, (Vector3){0,5,0}, GREEN);
                DrawLine3D((Vector3){0,0,0}, (Vector3){0,0,5}, BLUE);

                // Kutular
                for (int i = 0; i < 7; i++) {
                    DrawCube(kp[i], 0.8f,0.8f,0.8f, Fade(kr[i],0.7f));
                    DrawCubeWires(kp[i], 0.8f,0.8f,0.8f, kr[i]);
                }
            EndMode3D();

            // UI
            DrawRectangle(0,0,W,50, Fade(BLACK,0.7f));
            DrawText(TextFormat("Mod: %s  [1-4 ile degistir]", ad[mi]), 15, 15, 18, YELLOW);
            DrawText(TextFormat("pos=(%.1f,%.1f,%.1f) tgt=(%.1f,%.1f,%.1f)",
                kam.position.x, kam.position.y, kam.position.z,
                kam.target.x, kam.target.y, kam.target.z), 15, H-25, 12, GRAY);
        EndDrawing();
    }
    CloseWindow();
    return 0;
}

Önemli Noktalar

Mod Değişiminde Position Sıfırlama

Her mod, kamerayı farklı şekilde kontrol eder. FIRST_PERSON moduna geçerken position'ı {0,2,0} ve target'ı {0,2,1} yapıyoruz — göz seviyesinde, ileriye bakıyor. Yoksa kamera anlamsız bir yerde kalır.

DrawPlane vs DrawGrid

DrawPlane — düz yeşil zemin (dolu yüzey). DrawGrid — referans çizgiler (ızgara). İkisini birlikte kullanmak sahneye derinlik hissi verir.

🧪 Deneyin: ORBITAL Target Değiştirin

ORBITAL modunda kam.target'ı {3, 0, 3} yapın (2 numaralı satırda). Kamera artık orijin yerine o noktanın etrafında döner!

🧪 Deneyin: FOV 120°

FREE modunda kam.fovy = 120 yapın. Balık gözü efekti! Her şey bükülmüş ve garip görünür — ama çok geniş alan görürsünüz.

🧪 Deneyin: DisableCursor Kaldırın

DisableCursor() satırını yorum yapın (// ile). FREE modda fare pencere kenarına takılır ve kamera düzgün dönmez. Neden gerekli olduğunu anlarsınız.

🧪 Deneyin: FIRST_PERSON Boy Ayarı

kam.position.y = 0.5f yapın — çömelmiş gibi. = 5.0f yapın — dev gibi! Y değeri "göz yüksekliğiniz"dir.
