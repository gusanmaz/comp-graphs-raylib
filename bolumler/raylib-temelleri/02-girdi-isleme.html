Kullanıcıdan Girdi Almak

Bir programın interaktif olması için kullanıcıdan girdi (input) alması gerekir. Raylib'de girdi işleme çok basittir: her karede "şu tuş basılı mı?" diye sorarsınız. Buna polling (sorgulama) yöntemi denir.

Klavye Fonksiyonları

Raylib'de 4 temel klavye fonksiyonu var. Aralarındaki fark çok önemli:

FonksiyonNe Zaman true?Kullanım Alanı
IsKeyPressed(tus)Tuşa ilk basıldığı tek karedeAteş etme, menü seçimi, zıplama
IsKeyDown(tus)Tuş basılı tutulduğu sürece her kareSürekli hareket, koşma
IsKeyReleased(tus)Tuş bırakıldığı tek karedeOk bırakma, şarj bitirme
IsKeyUp(tus)Tuş basılı olmadığı süreceNadir kullanılır
ℹ️ Bilgi: Pressed vs Down Farkı

Düşünün: SPACE tuşuna 1 saniye basılı tutuyorsunuz (60 FPS'de 60 kare). IsKeyPressed sadece ilk karede true döner. IsKeyDown ise 60 kerede de true döner. Hareket için Down, tek seferlik eylem için Pressed kullanın.

Tuş Kodları

Sık kullanılan tuş sabitleri:

  • KEY_A - KEY_Z — harfler
  • KEY_ZERO - KEY_NINE — rakamlar
  • KEY_SPACE, KEY_ENTER, KEY_ESCAPE
  • KEY_UP, KEY_DOWN, KEY_LEFT, KEY_RIGHT — ok tuşları
  • KEY_LEFT_SHIFT, KEY_LEFT_CONTROL

Tipik Hareket Kodu

// Her karede kontrol: tus basili mi?
if (IsKeyDown(KEY_RIGHT)) konum.x += hiz;
if (IsKeyDown(KEY_LEFT))  konum.x -= hiz;
if (IsKeyDown(KEY_DOWN))  konum.y += hiz;
if (IsKeyDown(KEY_UP))    konum.y -= hiz;
🧪 Deneyin: IsKeyDown → IsKeyPressed

Yukarıdaki kodda IsKeyDown yerine IsKeyPressed yazın. Tuşa basılı tuttuğunuzda nesne sadece 1 piksel hareket eder — çünkü sadece ilk kare sayılır. Farkı hissedin!

Fare Girdisi

Fare Konumu

Vector2 fare = GetMousePosition();  // x ve y pixel koordinati
int fx = GetMouseX();  // sadece x
int fy = GetMouseY();  // sadece y

Fare Butonları

// Ayni mantik: Pressed, Down, Released
if (IsMouseButtonPressed(MOUSE_BUTTON_LEFT))  { /* tek tikla */ }
if (IsMouseButtonDown(MOUSE_BUTTON_LEFT))     { /* basili tut */ }
if (IsMouseButtonReleased(MOUSE_BUTTON_LEFT)) { /* birakildi */ }

Buton sabitleri: MOUSE_BUTTON_LEFT, MOUSE_BUTTON_RIGHT, MOUSE_BUTTON_MIDDLE

Fare Tekerleği

float tekerlek = GetMouseWheelMove();
// yukari cevirince +1, asagi cevirince -1, hareketsizse 0
// Zoom icin ideal:
zoom += tekerlek * 0.1f;

Gamepad (Kısa)

if (IsGamepadAvailable(0)) {  // 0 = ilk gamepad
    if (IsGamepadButtonPressed(0, GAMEPAD_BUTTON_RIGHT_FACE_DOWN)) {
        // Xbox A, PS X butonu
    }
    float solX = GetGamepadAxisMovement(0, GAMEPAD_AXIS_LEFT_X); // -1..+1
}

Örnek Kod: Girdi Demosu

🎮 Raylib: ornek08_girdi_isleme.c

Kutuyu WASD/oklar ile taşıyın, SPACE ile renk değiştirin, fare ile çizin.

#include "raylib.h"

int main(void)
{
    const int W = 900, H = 650;
    InitWindow(W, H, "Ornek 08 - Girdi Isleme");
    SetTargetFPS(60);

    Vector2 kutuPos = { 400.0f, 300.0f };
    float kutuBoyut = 40.0f;
    Color kutuRenk = BLUE;
    float kutuHiz = 4.0f;

    Vector2 noktalar[1024];
    int noktaSayisi = 0;
    float zoom = 1.0f;

    while (!WindowShouldClose())
    {
        // Klavye: basili tutma = surekli hareket
        if (IsKeyDown(KEY_RIGHT) || IsKeyDown(KEY_D)) kutuPos.x += kutuHiz;
        if (IsKeyDown(KEY_LEFT)  || IsKeyDown(KEY_A)) kutuPos.x -= kutuHiz;
        if (IsKeyDown(KEY_DOWN)  || IsKeyDown(KEY_S)) kutuPos.y += kutuHiz;
        if (IsKeyDown(KEY_UP)    || IsKeyDown(KEY_W)) kutuPos.y -= kutuHiz;

        // Klavye: tek basilma = bir kez tetikleme
        if (IsKeyPressed(KEY_SPACE)) {
            kutuRenk = (Color){ GetRandomValue(50,255),
                                GetRandomValue(50,255),
                                GetRandomValue(50,255), 255 };
        }

        // Sinir kontrolu
        if (kutuPos.x < 0) kutuPos.x = 0;
        if (kutuPos.y < 0) kutuPos.y = 0;
        if (kutuPos.x > W - kutuBoyut) kutuPos.x = W - kutuBoyut;
        if (kutuPos.y > H - kutuBoyut) kutuPos.y = H - kutuBoyut;

        // Fare: sol buton basili tutulursa ciz
        Vector2 farePos = GetMousePosition();
        if (IsMouseButtonDown(MOUSE_BUTTON_LEFT) && noktaSayisi < 1024) {
            noktalar[noktaSayisi] = farePos;
            noktaSayisi++;
        }

        // Fare tekerlegi: zoom
        zoom += GetMouseWheelMove() * 0.1f;
        if (zoom < 0.2f) zoom = 0.2f;
        if (zoom > 3.0f) zoom = 3.0f;

        BeginDrawing();
            ClearBackground(RAYWHITE);

            // Kutu (zoom ile boyutu degisir)
            float boyut = kutuBoyut * zoom;
            DrawRectangle((int)kutuPos.x, (int)kutuPos.y,
                          (int)boyut, (int)boyut, kutuRenk);

            // Fare ile cizilen noktalar
            for (int i = 0; i < noktaSayisi; i++) {
                DrawCircleV(noktalar[i], 3.0f, RED);
                if (i > 0) DrawLineV(noktalar[i-1], noktalar[i], Fade(RED, 0.4f));
            }
        EndDrawing();
    }
    CloseWindow();
    return 0;
}

Önemli Satırların Açıklaması

Vector2 noktalar[1024];

Dizi (array) kavramı: 1024 adet Vector2 tutabilen bir kutu. noktalar[0] ilk eleman, noktalar[1023] son eleman. C'de dizi boyutunu baştan belirtmeniz gerekir — bu yüzden 1024 gibi "yeterince büyük" bir sayı seçiyoruz.

GetRandomValue(50, 255)

50 ile 255 arasında rastgele tam sayı üretir. Her SPACE basışında farklı bir renk oluşur.

IsMouseButtonDown ile çizim

Down kullandığımız için fare sol tuşu basılı tutulduğu sürece her karede yeni bir nokta eklenir. Bu da çizgi çizme efekti yaratır.

🧪 Deneyin: Hız Değeri

kutuHiz değerini 1.0f ve 12.0f yaparak farkı hissedin. Düşük değer = hassas kontrol, yüksek değer = hızlı ama kaba hareket.

🧪 Deneyin: Sağ Tıkla Çizim

MOUSE_BUTTON_LEFT yerine MOUSE_BUTTON_RIGHT yazın. Artık sağ tıkla çizim yaparsınız.

🧪 Deneyin: Dizi Boyutunu Küçültün

noktalar[1024] yerine noktalar[50] ve sınır kontrolünü < 50 yapın. Sadece 50 nokta çizebilirsiniz — dizi taştığında program çökebilir!
