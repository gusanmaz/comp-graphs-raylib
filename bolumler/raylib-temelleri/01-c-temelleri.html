C Diline Hızlı Giriş

Bu bölümde sıfırdan başlayarak C dilinin temel yapılarını öğreneceksiniz. Daha önce Python, JavaScript veya başka bir dil kullandıysanız, C'nin "düşük seviyeli" yapısı sizi şaşırtabilir — ama korkmayın. Raylib ile görsel programlar yazmak için ihtiyacınız olan her şeyi burada bulacaksınız.

Değişken Türleri

C dilinde her değişkenin bir tipi vardır. Tip, değişkenin bellekte ne kadar yer kaplayacağını ve hangi değerleri tutabileceğini belirler:

TipBoyutAçıklamaÖrnek
int4 byteTam sayıint x = 42;
float4 byteOndalıklı sayıfloat pi = 3.14f;
double8 byteÇift hassas ondalıkdouble g = 9.81;
char1 byteTek karakterchar h = 'A';
bool1 bytetrue / falsebool ok = true;
ℹ️ Bilgi: float Sonundaki "f"

C'de 3.14 yazdığınızda derleyici bunu double olarak anlar. Eğer float bir değişkene atıyorsanız, 3.14f yazarak "bu bir float" demiş olursunuz. Raylib'de hemen her yerde float kullanılır, bu yüzden sayıların sonuna f koymayı alışkanlık edinin.

Değişken Tanımlama

C'de değişken tanımlamanın formülü şudur:

tip isim = deger;

Örnekler:

int skor = 0;           // tam sayi
float hiz = 2.5f;       // ondalikli
bool oyunBitti = false;  // dogru/yanlis
char seviye = 'A';       // tek karakter
🧪 Deneyin: Tip Değiştirme

float hiz = 2.5f; satırını int hiz = 2; olarak değiştirin. Ondalık kısmı kaybolur — çünkü int sadece tam sayı tutar.

struct (Yapı) Kavramı

Birden fazla değişkeni bir arada tutmak istediğinizde struct kullanırsınız. Bir "kutu" düşünün — içinde farklı türde şeyler var:

// Bir daireyi tanimlayan yapi
typedef struct {
    float x;        // merkez X
    float y;        // merkez Y
    float yaricap;  // yaricap
    Color renk;     // renk (bu da bir struct!)
} Daire;

typedef struct { ... } Daire; — Bu kalıp "yeni bir tip oluştur ve adına Daire de" anlamına gelir. Artık Daire kelimesini int veya float gibi kullanabilirsiniz:

Daire d = { 100.0f, 200.0f, 30.0f, RED };
// Alanlara erisim: nokta (.) ile
d.x = 150.0f;
d.yaricap = 50.0f;

Raylib'in Temel Veri Tipleri

Raylib kütüphanesi kendi struct'larını sağlar. Bunları tanımlamanız gerekmez — #include "raylib.h" dediğinizde otomatik gelirler.

Vector2 — 2D Konum

// Vector2 icinde sadece x ve y var
Vector2 konum = { 400.0f, 300.0f };
konum.x += 5.0f;  // saga tasi

Vector2 her yerde kullanılır: nesne konumu, hız, yön, fare pozisyonu...

Vector3 — 3D Konum

// 3D uzayda bir nokta: x, y, z
Vector3 pos = { 1.0f, 2.0f, 3.0f };
pos.y += 0.5f;  // yukari tasi

Color — Renk (RGBA)

// 4 deger: Kirmizi, Yesil, Mavi, Alpha (saydamlik)
// Her biri 0-255 arasi
Color turuncu = { 255, 165, 0, 255 };  // tam opak turuncu
Color yarisaydam = { 255, 0, 0, 128 }; // %50 saydam kirmizi

Raylib hazır renkler sunar: RED, GREEN, BLUE, WHITE, BLACK, YELLOW, PURPLE, ORANGE, RAYWHITE, DARKGRAY, LIGHTGRAY...

ℹ️ Bilgi: Fade() Fonksiyonu

Fade(RED, 0.5f) — kırmızı rengin alpha değerini %50 yapar (yarı saydam). 0.0f = tamamen görünmez, 1.0f = tamamen opak.

Rectangle — Dikdörtgen

// x, y, genislik, yukseklik
Rectangle kutu = { 50, 100, 200, 80 };
// x=50, y=100 noktasindan baslayan, 200 genis, 80 yuksek dikdortgen

Raylib Program Yapısı

Her raylib programının iskeleti şöyledir:

#include "raylib.h"   // raylib fonksiyonlarini dahil et

int main(void)
{
    InitWindow(800, 600, "Pencere Basligi");  // pencere ac
    SetTargetFPS(60);  // saniyede 60 kare

    while (!WindowShouldClose())   // ESC veya X basilana kadar
    {
        // --- GUNCELLEME ---
        // Degiskenleri degistir, fizik hesapla...

        // --- CIZIM ---
        BeginDrawing();
            ClearBackground(RAYWHITE);  // ekrani temizle
            // Burada cizim yap...
        EndDrawing();
    }

    CloseWindow();  // pencereyi kapat, kaynaklari serbest birak
    return 0;
}
ℹ️ Bilgi: SetTargetFPS(60)

Bu fonksiyon "saniyede en fazla 60 kare çiz" der. Neden? Ekranlar genelde 60Hz'dir. 60 FPS üstü gereksiz iş yükü yaratır. Ayrıca oyun mantığı tutarlı kalır.

ℹ️ Bilgi: while Döngüsü

while (!WindowShouldClose()) — "pencere kapatılmadığı sürece devam et" demek. Her dönüşte 1 kare (frame) çizilir. 60 FPS = saniyede 60 kez döner.

Örnek Kod: Tüm Tipleri Görelim

🎮 Raylib: ornek07_c_temelleri.c

Bu program tüm temel C tiplerini ve raylib veri yapılarını ekranda canlı olarak gösterir.

#include "raylib.h"
#include <math.h>

typedef struct {
    float x;
    float y;
    float yaricap;
    Color renk;
} Daire;

int main(void)
{
    int sayac = 0;
    float hiz = 2.5f;
    bool aktif = true;

    Vector2 konum2d = { 400.0f, 300.0f };
    Vector3 konum3d = { 1.0f, 2.0f, 3.0f };
    Color   renk    = { 255, 100, 50, 255 };
    Rectangle kutu  = { 50, 400, 200, 80 };

    Daire daire1 = { 600.0f, 250.0f, 40.0f, BLUE };
    Daire daire2 = { 300.0f, 450.0f, 25.0f, GREEN };

    InitWindow(900, 600, "Ornek 07 - C Temelleri");
    SetTargetFPS(60);

    while (!WindowShouldClose())
    {
        sayac++;
        Vector2 fare = GetMousePosition();
        konum2d.x += (fare.x - konum2d.x) * 0.02f;
        konum2d.y += (fare.y - konum2d.y) * 0.02f;
        renk.g = (unsigned char)(128 + 127 * sinf(sayac * 0.03f));
        renk.b = (unsigned char)(128 + 127 * cosf(sayac * 0.03f));
        daire1.x = 600.0f + sinf(sayac * 0.05f) * 100.0f;

        BeginDrawing();
            ClearBackground(RAYWHITE);
            DrawText("C Temelleri ve Raylib Veri Tipleri", 20, 15, 24, DARKGRAY);

            DrawText(TextFormat("int   sayac = %d", sayac), 20, 60, 18, DARKBLUE);
            DrawText(TextFormat("float hiz   = %.1f", hiz), 20, 85, 18, DARKBLUE);
            DrawText(TextFormat("bool  aktif = %s", aktif ? "true" : "false"), 20, 110, 18, DARKBLUE);

            DrawCircleV(konum2d, 12.0f, RED);
            DrawText(TextFormat("Vector2 = { %.0f, %.0f }", konum2d.x, konum2d.y), 20, 145, 16, MAROON);

            DrawRectangle(20, 245, 60, 30, renk);
            DrawText(TextFormat("Color = { %d, %d, %d, %d }", renk.r, renk.g, renk.b, renk.a), 90, 250, 16, DARKGRAY);

            DrawRectangleRec(kutu, Fade(SKYBLUE, 0.5f));
            DrawRectangleLinesEx(kutu, 2, DARKBLUE);

            DrawCircle((int)daire1.x, (int)daire1.y, daire1.yaricap, daire1.renk);
            DrawCircle((int)daire2.x, (int)daire2.y, daire2.yaricap, daire2.renk);
        EndDrawing();
    }
    CloseWindow();
    return 0;
}

Kodun Satır Satır Açıklaması

#include <math.h>

Matematik fonksiyonlarını kullanmak için: sinf() (sinüs) ve cosf() (kosinüs). Bunlar -1 ile 1 arası değer döndürür ve renk animasyonu için kullanıyoruz.

typedef struct { ... } Daire;

Kendi veri tipimizi tanımlıyoruz. İçinde 4 alan var: x, y (konum), yaricap (büyüklük), renk (görünüm). Bu sayede bir daireyle ilgili tüm bilgiyi tek bir değişkende tutabiliriz.

konum2d.x += (fare.x - konum2d.x) * 0.02f;

Bu satır lineer interpolasyon (lerp) yapar. Fare ile konum2d arasındaki farkın %2'si kadar hareket eder — bu da yumuşak bir takip efekti oluşturur.

(unsigned char)(128 + 127 * sinf(...))

sinf() -1 ile 1 arası değer verir. 128 + 127 * sin = 1 ile 255 arası. (unsigned char) ile float'ı 0-255 arası tam sayıya dönüştürüyoruz — çünkü Color'ın alanları unsigned char tipindedir.

TextFormat("int sayac = %d", sayac)

TextFormat raylib'in printf benzeri fonksiyonudur. %d = tam sayı, %.1f = 1 ondalıklı float, %s = metin. Sonucu ekrana yazdırmak için DrawText ile kullanılır.

🧪 Deneyin: Lerp Hızını Değiştirin

0.02f değerini 0.1f yapın — kırmızı daire fareye çok daha hızlı takip eder. 0.005f yapın — çok yavaşlar. Bu "yumuşak takip" oyunlarda çok kullanılır.

🧪 Deneyin: Renk Animasyon Hızı

sinf(sayac * 0.03f) içindeki 0.03f değerini 0.1f yapın — renk çok hızlı değişir. 0.005f yapın — çok yavaş ve zarif bir geçiş olur.

🧪 Deneyin: Kendi Struct'ınızı Ekleyin

Bir Kare struct'ı tanımlayın: typedef struct { float x, y, boyut; Color renk; } Kare; — sonra DrawRectangle ile ekrana çizdirin.

Özet

  • int, float, bool — temel C tipleri
  • struct — birden fazla değeri bir arada tutar
  • Vector2, Vector3 — konum ve yön için
  • Color — RGBA renk (0-255)
  • Rectangle — dikdörtgen alanı
  • InitWindow → while → EndDrawing → CloseWindow — program iskeleti
